# PowerBI_Projects_2026

## Сквозная аналитика План-Факт для производственного холдинга (Power BI)

1. Описание проекта
- Разработка интерактивной панели управления (Dashboard) для мониторинга выполнения плана продаж в разрезе менеджеров и бизнес-направлений (Стройка, Мебель, Транспорт).
- Цель: Синхронизация данных плана (CRM) и факта (1С) для принятия оперативных решений и прогнозирования на 2026 год.
3. Стек технологий - Power BI Desktop
- Power Query (M): Автоматизация сбора данных, очистка данных (ETL), нормализация справочников, обработка пропущенных значений (null) и дубликатов.
- DAX: Реализация сложных мер, работа с виртуальными связями, контекстом фильтрации и Time Intelligence.
3. Архитектура модели: Реализована схема «Снежинка»
- Внедрен Master-календарь для синхронизации временных шкал из разных систем (CRM и 1С).
- Сквозная фильтрация: Использована функция USERELATIONSHIP для активации неактивных связей между CRM-сделками и фактическими продажами.
4. Ключевые показатели и визуализация (KPIs)
- План-фактный анализ 2025: Отображение процента выполнения плана в реальном времени.
- Прогноз 2026: Сегментация подтвержденных заказов на будущий период.
- Анализ направлений: Визуализация структуры выручки по категориям.
5. Аналитический аудит данных 
В ходе проекта выявлен и устранен ряд проблем в исходных данных:
- Разрыв сквозной аналитики из-за отсутствия уникальных ID заказов в части CRM-сделок.
- Неполнота справочников (нормализованы через логику «Прочие» для сохранения целостности сумм).
